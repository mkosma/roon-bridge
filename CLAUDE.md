# Claude Instructions for Roon Bridge

## Default Zone

Always use the **"WiiM + 1"** zone for all Roon commands unless the user expressly asks for another device or zone.

## Adding new MCP endpoints

When adding a new tool to the MCP server, also add Claude Code permission entries under both registered prefixes (local and claude.ai connector). See [docs/adding-endpoints.md](docs/adding-endpoints.md) for the procedure.
